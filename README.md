# Flappy Bird

A mobile-optimized Flappy Bird clone built with vanilla HTML5 Canvas and JavaScript.

## Play

Open `index.html` in any browser or visit the hosted version.

## Features

- Touch controls optimized for mobile
- Smooth physics and animations  
- Classic gameplay mechanics
- No dependencies - single HTML file

## Controls

- **Tap/Click**: Flap to fly
- **Space**: Flap (keyboard)
